### 질문거리

#### get '/boards' do : 샘 자료

```ruby
get '/boards' do
  @boards = []
  CSV.open('./boards.csv', 'r').each do |row|
    @boards << row
  end
  erb :boards
end
```

#### get '/boards' do : 내 자료

```ruby
get '/boards' do
  @boards = []                                  # 1
  CSV.open('./boards.csv', 'r').each do |row|  # 2  현재 폴더에 있는 csv 파일을 열겠다.
    @boards << row        
  end
  erb :boards
  puts @boards
end
```

차이점 :   

```ruby
  erb :boards
  puts @boards			작동 안함.

  puts @boards
  erb :boards            정상 작동
```
